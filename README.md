# Friend Portal
just for fun :) see demo http://fportal-yacneyac.rhcloud.com/

- Python 2.6
- Tornado 4
- MySQL 5.5
- WebSocket
- Angular 1.4.7
- Bootstrap 3.3.5

USER
* auth by facebook, twitter or create local user
* save own files
* create own event (sync calendar with FB)
* add friends
